# AI 博主工作台初始框架

这份文件记录你的第一版需求和后续迭代方向，后续改项目时可作为基准。

## 你的核心定位

你是一名抖音和小红书 AI 博主，现阶段主要做 AI 科普视频。内容来源包括 X、Reddit 和 YouTube 上近期爆款 AI 相关视频或讨论，经过翻译、理解、重写和中文化表达后，传播给中文用户做知识科普。

## 已纳入网站的菜单

1. 任务计划
   - 我的计划：日计划、周计划、月计划、季度计划、年计划
   - 任务管理：记录任务，并标记为待办、进行中或已完成
2. Skills 管理
   - 记录自制 Skill、他人 Skill、待研究 Skill
3. 后台管理
   - 账号定位、平台档案、SOP、常用后台信息
4. 全部入口
   - 归纳未来新增的网站入口、工具入口、资料入口
   - 第二版已加入默认入口：素材版权与来源记录、发布日历与平台适配、账号数据看板、Prompt / 模板库、选题评分机制
5. 选题合集库
   - 收集 X、Reddit、YouTube 上最近爆款 AI 视频或帖子
6. 写作室
   - 记录个人想法、脚本草稿、标题、钩子和表达结构
7. 经验复盘
   - 记录内容发布后的表现、经验和下次优化方向

## 第二版新增

1. 全局 UI 风格升级
   - 改为淡粉色和白色的清爽系配色
   - 提升卡片阴影、圆角、按钮和输入框精致度
2. 文本修改
   - 左上角标题改为“用户的工作台”
   - 左下角用户栏改为“用户”
3. 账号登录系统
   - 新增登录界面：用户名、账号、密码
   - 登录后左下角动态显示用户名
4. 同步功能
   - 新增“同步”按钮
   - 点击后显示 loading 动画和同步完成提示
   - 第三版已改为 Supabase Auth + 数据库云端同步：GitHub Pages 负责展示网页，Supabase 负责账号登录和保存数据
   - 电脑和手机使用同一邮箱账号与密码登录后，会读写同一个 Supabase 用户名下的工作台数据
5. 移动端适配
   - 手机端导航栏自动折叠隐藏
   - 顶部显示菜单按钮，点击后展开侧边栏
   - 内容区域自动适配手机宽度

## 后续建议

1. 接入真实账号系统
   - 可使用 Supabase、Firebase、自建后端或其他云服务。
2. 接入真实云端数据库
   - 已提供 `supabase_workbench_sync.sql` 建表文件，包含 Row Level Security，只允许登录用户读写自己的数据。
   - 需要在 HTML 顶部脚本配置中填写 `SUPABASE_URL` 和 `SUPABASE_ANON_KEY`，再重新上传 GitHub Pages。
3. 把“全部入口”的默认模块做成独立页面
   - 素材版权与来源记录
   - 发布日历与平台适配
   - 账号数据看板
   - Prompt / 模板库
   - 选题评分机制
4. 增加内容生产流水线
   - 收集、筛选、翻译、重写、成稿、剪辑、发布、复盘、归档。
5. 增加风险检查清单
   - 检查事实准确性、版权风险、平台敏感词、夸大表达和误导性表达。
